Column Guide

issue fields
Title is the task name. Owner, priority, due, and week are stored as issue body fields. Labels like weekly-tracker, sreerag, nandana, work, and personal make filtering easier if they exist in GitHub.
Done means the GitHub issue is closed. Open means pending. External ticket URLs can go in the notes and will stay visible to everyone.
